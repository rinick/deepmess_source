Tsuro Chess is a board game base on [Tsuro: The Game of the Path](https://calliopegames.com/shop/tsuro-the-game-of-the-path/)

The following rules requires some knowledge of the original Tsuro game.


### Setup

You will need the game board from the original Tsuro game as well as all the 35 path tiles and the dragon tile.

##### 2 players
2 different colors of pieces are needed, you will need 3 pawn pieces and 1 king piece for each color.

To start the game:
1. Lay out the board.
1. Draw a configuration card, flip it up to see which color each player uses and the position of pieces on the board.
  - Red always plays first.
  - Since the red and blue arrow will assign player\'s color, configuration card\'s back side should be centrally-symmetric.
1. Players place pieces based on configuration card, player can choose any start mark that belongs to the tile.
1. Shuffle the path tiles and deal 2 tiles to each player, players\' tiles should always be visible to other players.

##### 3 ~ 6 players

Each player has 1 king and 2 pawns instead of 3 pawns: ♔♙♙<br/>
Among all the 36 squares, the 20 squares next to the edge need to have labels or numbers on them: E1~E4 W1~W4 S1~S4 N1~N4 SE SW NE NW  or simply just 1~20<br/>
There should also be a D20 dice with the corresponding labels on it.<br/>
No need for configuration cards.<br/>


To start the game:
1. Lay out the board.
1. Each player choose a color, and decide the order, it will be the order of placing pieces as well as the order of playing path tiles.
1. Players place pieces on the start marks in turn. To do so, the player first roll the D20 dice, find the t
  - Pieces can not be put on a tile that already has opponent\'s piece.
  - Pieces can be put on a tile that already has your own piece.
1. Shuffle the path tiles and deal 2 tiles to each player, players\' tiles should always be visible to other players.

### Play the game

Each Turn has three parts:
1. Deal a path tile to player.
2. Player plays a path tile on a open square next to any of the player\'s pieces.
3. Move all the pieces affected by the tile and remove any piece that goes out of board.

##### Winning the game

- When a player\'s king is eliminated, player\'s other pieces will also be eliminated from board. His/her path tile will be given to other players.
- If a player has no other choice but eliminate his/her own king, the player loses immediately.
  - Path tile is not allowed to be played in this case, which also means the player\'s remaining pieces are directly removed from board and other player\'s pieces are not moved during this turn.
- When only one king piece remains on the board, that player wins the game.
- If more than one king remains after all 35 tiles are played, it\'s a draw.
  - In case there must be a winner, the player that plays the last path tile wins.
